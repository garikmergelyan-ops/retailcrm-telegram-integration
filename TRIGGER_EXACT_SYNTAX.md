# ✅ ТОЧНЫЙ СИНТАКСИС для "custom query body"

## 📋 Что вставлять в поле "custom query body"

### ❌ НЕПРАВИЛЬНО (без фигурных скобок):

```
"order": {{ order|json_encode }},
"event": "orderStatusChange"
```

### ✅ ПРАВИЛЬНО (с фигурными скобками):

```
{
  "order": {{ order|json_encode }},
  "event": "orderStatusChange"
}
```

## 📝 Пошаговая инструкция

1. Откройте триггер для редактирования
2. Найдите раздел **"Send parameters"**
3. Выберите **"custom query body"**
4. В поле вставьте **ТОЧНО** такой текст (БЕЗ слова "json" и БЕЗ обратных кавычек):

```
{
  "order": {{ order|json_encode }},
  "event": "orderStatusChange"
}
```

## ⚠️ ВАЖНО

- **Должны быть фигурные скобки** `{ }` в начале и конце
- **Должны быть кавычки** вокруг ключей `"order"`, `"event"`
- **НЕ должно быть слова "json"** перед текстом
- **НЕ должно быть обратных кавычек** ``` вокруг текста
- **Пробелы важны:** `{{ order|json_encode }}` (с пробелами), а НЕ `{{order|json_encode}}`

## 🔍 Как должно выглядеть в поле:

```
┌─────────────────────────────────────┐
│ custom query body                  │
├─────────────────────────────────────┤
│ {                                   │
│   "order": {{ order|json_encode }},│
│   "event": "orderStatusChange"      │
│ }                                   │
└─────────────────────────────────────┘
```

## ✅ Альтернативный вариант (если первый не работает):

Если `{{ order|json_encode }}` не работает, попробуйте:

```
{
  "order": {{ order }},
  "event": "orderStatusChange"
}
```

Или просто:

```
{{ order|json_encode }}
```
