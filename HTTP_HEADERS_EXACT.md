# ✅ ТОЧНЫЙ СИНТАКСИС для HTTP заголовков в триггере

## 📋 Как добавить HTTP заголовок

### Шаг 1: Нажмите "Add header"

В разделе **"HTTP-headers"** нажмите кнопку **"Add header"** (или **"Добавить заголовок"**)

### Шаг 2: Введите имя заголовка

В поле **"Header name"** (или **"Имя заголовка"**) введите:

```
Content-Type
```

**БЕЗ кавычек!** Просто: `Content-Type`

### Шаг 3: Введите значение заголовка

В поле **"Header value"** (или **"Значение заголовка"**) введите:

```
application/json
```

**БЕЗ кавычек!** Просто: `application/json`

## ✅ Правильно:

```
┌─────────────────────────────────────┐
│ HTTP-headers                       │
├─────────────────────────────────────┤
│ Header name:  Content-Type          │
│ Header value: application/json      │
└─────────────────────────────────────┘
```

## ❌ Неправильно:

```
┌─────────────────────────────────────┐
│ HTTP-headers                       │
├─────────────────────────────────────┤
│ Header name:  "Content-Type"       │  ← НЕ НУЖНЫ кавычки!
│ Header value: "application/json"   │  ← НЕ НУЖНЫ кавычки!
└─────────────────────────────────────┘
```

## 📝 Пример заполнения полей:

1. **Header name:** `Content-Type` (вводите БЕЗ кавычек)
2. **Header value:** `application/json` (вводите БЕЗ кавычек)
3. Нажмите **"Add"** или **"Сохранить"**

## ⚠️ ВАЖНО:

- **НЕ используйте кавычки** вокруг значений
- **НЕ используйте обратные кавычки** ```
- Просто введите текст как есть: `Content-Type` и `application/json`

---

**После добавления:** Заголовок должен появиться в списке заголовков. Сохраните триггер!
